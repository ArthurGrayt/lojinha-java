# 🛒 Loja Interativa em Java

Projeto simples desenvolvido em Java para simular uma loja interativa no console. Feito como prática de desenvolvimento e aprimoramento dos conceitos de Java.

## 🚀 Funcionalidades

- Registro do nome do usuário com validação
- Menu de categorias:
  - 🎮 Games
  - 🕹️ Consoles
  - 🤖 Action Figures
  - 👕 Roupas

- Mensagens dinâmicas de acordo com a escolha

## 💻 Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- IDE: IntelliJ IDEA
